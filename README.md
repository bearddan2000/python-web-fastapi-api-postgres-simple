# python-web-fastapi-api-postgres-simple

## Description
Simple web app that serves api
for a fastapi project.

Uses sqlalchemy query a table `dog`.

## Tech stack
- python
  - fastapi
  - uvicorn
  - sqlalchemy
- bootstrap
- jquery
- dataTable
- postgresql

## Docker stack
- python:latest
- postgresql

## To run
`sudo ./install.sh -u`
- [Availble at](http://localhost/dogs)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
